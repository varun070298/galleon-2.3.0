Music Player

Player: Choose the classic HMO-like player or the Winamp player.
Winamp Classic Skin: The skin to be used for the Winamp player.
Random play folders: Enable random play of music when playing folders.
Screensaver: Enable the player screensaver. The screensaver will automatically dim the screen after 5 minutes.
User Amazon.com: If the MP3 does not have embedded album art, the art can be downloaded from Amazon.com.
User Folder.jpg: If the MP3 does not have embedded album art, the file Folder.jpg will be used instead.
Show web images: Allow the display of web images about the current MP3 artist.
